# Prolabels Dashboard MVP

## Instalación rápida (necesitas Node.js 18+)

```bash
npm install
npm run dev
```

Abrir: http://localhost:5173

## Uso
1. El sidebar izquierdo siempre está visible con 3 iconos: Inicio 🏠, Costos 💰, Dashboard 📊
2. Carga tus Excel desde la pantalla de inicio
3. Al cargar el Excel de costos → se navega automáticamente a la vista Costos
4. Al cargar el Reporte Valorizado → disponible en Dashboard
5. Clic en filas de la tabla para filtrar todos los gráficos
6. Selector desplegable en la barra superior para cambiar presupuesto
